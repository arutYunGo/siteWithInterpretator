# siteWithInterpretator
На сайте присутствует контейнер блоков и рабочая зона, в которой можно работать с этими самыми блоками.
Чтобы начать писать свою программу, нужен обязательно блок Start Programm, в который можно вкладывать другие блоки,
если это разрешено(например: блок присваивания можно, а просто блок сложения нельзя)
Блоки математических операций, логики, и литералов можно вкладывать только в ветки других блоков.
Абсолютно каждое действие зависит от блоков, не считая блоков, в которых есть input, ведь надо вписывать вручную.
В правом контейнере находится консоль, в которую можно что-то выводить,благодаря print, который будет так же находиться
в блоке start, либо, если ошибка в программе, то будет выводиться название ошибки в консоли и программа не выполнится.
Присутсвует сохранение, загрузка уже готовых алгоритмов.
Присутсвует панинг и зумирование
